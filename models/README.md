# Model Files

Pre-trained models for AstroPredict Phase-1.

| File | Description | Size |
|------|-------------|------|
| lstm_phase1_model.keras | LSTM temporal model | ~1.7MB |
| bilstm_phase1_model.keras | BiLSTM temporal model | ~1.9MB |
| sharp_context.pkl | SHARP context library | ~0.88MB |

These models were trained on 1.36M observations across 
309 solar active regions using HARPNUM-level data splitting.
